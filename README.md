## Karvio App (Expo React Native)

This is a React Native application created with Expo using the TypeScript blank template.

### Getting started

- **Install dependencies**

```bash
cd karvio-app
npm install
```

- **Run the development server**

```bash
npm run start
```

Then use one of:

- **Android**: `npm run android` (Android emulator or Android device with Expo Go)
- **iOS**: `npm run ios` (requires macOS and Xcode) or use Expo Go on a physical device
- **Web**: `npm run web`

### Project structure

- `App.tsx`: Main application component.
- `assets/`: Application icons and images.
- `index.ts`: Entry point used by Expo.

### Useful commands

- **Start Metro bundler**: `npm run start`
- **Android**: `npm run android`
- **iOS**: `npm run ios`
- **Web**: `npm run web`

### Notes

- This project uses:
  - **Expo** `~54.x`
  - **React Native** `0.81.x`
  - **React** `19.x`
  - **TypeScript** `~5.9.x`
